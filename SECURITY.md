# Security

If you discover a vulnerability, please report it responsibly.

## Contact

Email: dev@vaultum.app

## Scope

- `src/AgentBondManager.sol`
- `src/ReputationScorer.sol`
- `src/interfaces/`

## Out of Scope

- Test files and mocks
- Third-party dependencies (OpenZeppelin)

## Disclosure

Please allow 72 hours before public disclosure. We will acknowledge receipt within 24 hours.
